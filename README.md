## 3. 서론

### 배경
초미세먼지(PM2.5)는 국민 건강 및 사회적 비용과 직결되는 중요한 환경 지표이다. 단기적 농도 예측은 취약계층 보호, 공공 경보, 환경 정책 수립에 기여할 수 있다. 가스 성분(NO2, O3, CO, SO2 등)은 PM2.5 형성에 영향을 주므로 이들만으로도 예측 가능한 모델 구축을 탐색한다.

### 프로젝트 목표

- `Ridge`와 `Random Forest` 모델의 성능 비교 및 해석
- 시계열 특성(계절성·시차) 반영한 피처 엔지니어링과 시간 기반 검증 적용

## 4. 본론
프로젝트 수행 과정 및 적용 알고리즘을 상세히 기술한다. 표·다이어그램·그래프를 적극 활용.

### 4.1 관련 이론 및 기술
- **Ridge 회귀**: L2 정규화를 통한 안정적 선형 회귀. 특성 간의 다중공선성을 완화하고 가중치가 과도하게 커지는 것을 방지하여 모델을 안정화한다.
- **Lasso 회귀**: L1 정규화를 사용하여 가중치의 희소성(sparsity)을 유도한다. 모델 예측에 불필요한 변수의 가중치를 0으로 만들어 자연스럽게 변수 선택(Feature Selection)이 이루어지며 모델 해석성이 높다.
- **Random Forest 회귀**: 다수의 결정 트리(Decision Tree)를 앙상블하여 비선형적인 데이터 패턴과 상호작용을 파악하는 머신러닝 알고리즘이다. 
    - 배깅(Bagging) 기법을 사용하여 훈련 데이터에서 무작위 샘플링을 통해 여러 트리를 독립적으로 학습시키고, 그 결과를 평균 내어 단일 트리의 단점인 과적합(Overfitting)을 방지한다.
    - **주요 하이퍼파라미터 3가지**: 모델의 성능과 복잡도를 조절하기 위해 다음 세 가지를 적절히 설정해야 한다.
        1) **트리 수 (Number of trees):** 앙상블에 포함할 전체 결정 트리의 개수(`n_estimators`). 트리가 많을수록 예측이 안정되지만 계산 비용이 증가한다.
        2) **노드 크기 (Node size):** 트리가 얼마나 깊게 자랄지를 제한하는 기준. 최대 깊이(`max_depth`)나 노드 분할을 위한 최소 샘플 수 등을 설정하여 모델의 과적합을 제어한다.
        3) **샘플링된 특징 수 (Number of sampled features):** 트리의 각 분기(가지치기) 과정에서 분할 기준으로 고려할 무작위 피처(특성)의 수(`max_features`). 트리의 다양성을 확보하는 데 중요한 역할을 한다.

### 4.2 데이터 설명 및 전처리

본 프로젝트에서는 대한민국 공공데이터 포털의 대기오염 관측 데이터(`airkorea_2024_seongbuk.csv`)를 사용하였다.
데이터에는 측정일시에 따른 여러 가스와 미세먼지 농도 등의 정보가 포함되어 있다.
    :

전처리 과정으로 다음 작업을 수행하였다.
- 파생 변수(월, 시간대) 추출
    : 문자열 형태의 `측정일시` 데이터를 바탕으로 모델이 학습할 수 있도록 정수형의 `월`과 `시간대` 특성을 추출하였다. 24시는 0시로 변환 처리하였다.
- 불필요한 변수 제거
    : 모델 예측과 무관한 식별 데이터(지역, 망, 측정소코드, 측정소명, 주소 등)와 타겟과 직결되는 변수(미세먼지(PM10))를 분석에서 제외하였다.
- 결측치 제거
    : 결측값이 하나라도 포함된 데이터 행은 모두 제거(`dropna()`)하여 완전한 데이터를 구성하였다.

### 4.3 모델 설계 및 실험 방법

본 프로젝트에서는 초미세먼지 농도를 예측하기 위해 세 가지 다른 회귀 모델(Ridge, Lasso, Random Forest)을 설계하여 비교 실험을 진행하였다.

**1. 실험 환경 및 공통 설정**
- 데이터 분할 (Train/Test Split)
    : 전체 데이터 중 80%를 모델 학습(Train)에 사용하고, 나머지 20%를 보이지 않는 데이터에 대한 일반화 성능 평가(Test)에 사용하기 위해 8:2 비율로 분할하였다. 결과의 재현성을 위해 시드(`random_state=42`)를 고정하였다.
- 데이터 스케일링 (Data Scaling)
    : 선형 회귀 계열(Ridge, Lasso) 모델은 변수의 스케일에 민감하므로 데이터를 정규화(예: StandardScaler 적용)한 후 학습을 진행하였다.
- 성능 평가 지표 (Metrics)
    : 모델의 예측 성능을 측정하기 위해 MAE(평균 절대 오차), RMSE(루트 평균 제곱 오차), R2-Score(결정 계수)를 주요 지표로 활용하여 평가하였다.

**2. 모델별 설계 및 하이퍼파라미터 설정**
- **Ridge 회귀 모델 (L2 정규화)**
    : 특성 간의 다중공선성을 완화하고 모델의 안정성을 높이기 위해 L2 패널티를 부여하는 Ridge 회귀를 적용하였다. 규제 강도 파라미터(alpha)를 조절하여 과적합을 방지하면서도 안정적인 예측을 수행하도록 설정하였다.
- **Lasso 회귀 모델 (L1 정규화)**
    : 불필요한 변수의 가중치를 0으로 만들어 의미 있는 변수만을 선택(Feature selection)하기 위해 L1 패널티를 부여하는 Lasso 회귀를 적용하였다. 규제 강도 파라미터(alpha)를 튜닝하여 주요 특성들이 결과에 미치는 영향을 확인하였다.
- **Random Forest 모델 (앙상블 트리)**
    : 비선형적인 시계열 데이터 패턴과 모델 간 상호작용을 효과적으로 학습하기 위해 Random Forest를 설계하였다. 트리의 개수를 100개(`n_estimators=100`)로 설정하였고, 학습 데이터에 과적합(Overfitting)되는 것을 방지하기 위해 트리의 최대 깊이는 10(`max_depth=10`)으로 제한하였다.

**3. 변수 중요도 분석**
- 앙상블 모델(Random Forest) 학습이 완료된 후 제공되는 변수 중요도(Feature Importance)와 선형 모델(Ridge, Lasso)의 회귀 계수 크기를 바탕으로, 초미세먼지 농도에 가장 큰 영향을 미치는 요인(가스 종류, 월, 시간대 등)을 파악하고 결과를 비교·시각화하였다.
